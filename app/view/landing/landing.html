Cfgram is an app that allows you to upload images and created galleries. Once you have sign up, you can create a gallery by providing a name and description. When you create multiple galleries, you can select which gallery to upload images to. For images, you can include a name and a description as well.

Cfgram was created by the teachers and students at Code Fellows.
